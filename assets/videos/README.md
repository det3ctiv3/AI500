# Demo Video Folder

## Upload Instructions

Upload your demo video to this folder with one of these names:
- `demo.mp4` (recommended - best compatibility)
- `demo.webm` (alternative format)
- `demo.ogg` (alternative format)

## Optional Files

- `thumbnail.jpg` - Video thumbnail/poster image (recommended size: 1280x720px)

## Supported Video Formats

- **MP4** (H.264 codec) - Best compatibility across all browsers
- **WebM** (VP8/VP9 codec) - Good for modern browsers
- **OGG** (Theora codec) - Alternative option

## Recommended Video Specifications

- **Resolution**: 1920x1080 (Full HD) or 1280x720 (HD)
- **Aspect Ratio**: 16:9
- **Codec**: H.264 for MP4
- **Bitrate**: 3-5 Mbps for 1080p, 1.5-3 Mbps for 720p
- **Duration**: 2-5 minutes (recommended for demo)
- **File Size**: Keep under 50MB for fast loading

## Quick Upload Command

From your local machine:
```bash
scp /path/to/your/demo.mp4 root@minzar.uz:/root/AI500/assets/videos/
```

## After Upload

The video will automatically appear on the website at:
- https://minzar.uz/demo
- https://minzar.uz/#demo

No code changes needed - just upload the file!
